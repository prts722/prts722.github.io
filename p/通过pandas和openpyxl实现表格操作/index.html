pandas和openpyxl的区别

在处理Excel文件时，Python提供了多个库，其中openpyxl和pandas是两个常用的选项。但使用场景各不相同

Openpyxl

Openpyxl是一个专门用于读写Excel 2010 xlsx/xlsm/xltx/xltm文件的库。它允许你操作Excel文件，例如创建新的工作表、读取或修改已有的工作表、更改单元格样式等。Openpyxl的优势在于它提供了对Excel文件结构的细粒度控制，特别是当需要处理复杂的Excel文件结构时，如合并单元格、设置公式等。

Pandas

Pandas是一个强大的数据分析和操作库，它提供了DataFrame和Series两种数据结构来处理表格数据。Pandas适合于数据清洗、转换、分析等任务，特别是在数据探索阶段，它能够快速地读取和转换数据，支持各种文件格式的输入输出，包括CSV、Excel等。 Pandas在处理大型数据集时表现出色，它可以快速读取和分析数据，但在处理复杂的Excel文件结构时可能不如openpyxl灵活。

性能和适用场景

在性能方面，openpyxl在读取大量数据时通常比pandas更快。例如，当处理稀疏矩阵数据集时，openpyxl显示出更高的效率。然而，pandas在数据分析方面更加强大，它提供了丰富的数据处理功能，如数据聚合、合并、分组等。在实际应用中，建议根据具体需求选择合适的库。如果需要处理复杂的Excel文件结构，或者需要对Excel文件的样式进行精细控制，openpyxl是更好的选择。而对于数据分析和探索，尤其是当数据结构相对简单时，pandas则更为合适。

结合使用

在某些情况下，结合使用openpyxl和pandas可以发挥两者的优势。例如，可以先使用openpyxl进行数据的预处理，如格式化和清理，然后再使用pandas进行深入的数据分析和探索。这种结合使用的方式可以提高工作效率，同时充分利用两个库的强项。openpyxl和pandas各有所长，选择哪个库取决于你的具体需求和工作流程。了解它们的特点和区别，可以帮助你更高效地处理Excel数据。

下面分别浅显的记录一下pandas和openpyxl的使用方法，在这之前都要通过pip install pandas/openpyxl来安装对应模块

Pandas

读取文件

Pandas 提供了丰富的 Excel 文件操作功能，帮助我们方便地读取和写入 .xls 和 .xlsx 文件，支持多表单、索引、列选择等复杂操作，是数据分析中必备的工具。

操作 方法 说明
读取 Excel 文件 pd.read_excel() 读取 Excel 文件，返回 DataFrame
将 DataFrame 写入 Excel DataFrame.to_excel() 将 DataFrame 写入 Excel 文件
加载 Excel 文件 pd.ExcelFile() 加载 Excel 文件并访问多个表单
使用 ExcelWriter 写多个表单 pd.ExcelWriter() 写入多个 DataFrame 到同一 Excel 文件的不同表单

通过pd.read_excel()读取 Excel 文件，它支持读取 .xls 和 .xlsx 格式的文件。

部分参数：

  • io：这是必需的参数，指定了要读取的 Excel 文件的路径或文件对象。
  • sheet_name=0：指定要读取的工作表名称或索引。默认为0，即第一个工作表。
  • header=0：指定用作列名的行。默认为0，即第一行。
  • names=None：用于指定列名的列表。如果提供，将覆盖文件中的列名。
  • index_col=None：指定用作行索引的列。可以是列的名称或数字。
  • usecols=None：指定要读取的列。可以是列名的列表或列索引的列表。
  • dtype=None：指定列的数据类型。可以是字典格式，键为列名，值为数据类型。
  • engine=None：指定解析引擎。默认为None，pandas 会自动选择。
  • converters=None：用于转换数据的函数字典。
  • true_values=None：指定应该被视为布尔值True的值。
  • false_values=None：指定应该被视为布尔值False的值。
  • skiprows=None：指定要跳过的行数或要跳过的行的列表。
  • nrows=None：指定要读取的行数。
  • na_values=None：指定应该被视为缺失值的值。
  • date_format=None：指定日期的格式。
  • skipfooter=0：指定要跳过的文件末尾的行数。

写入文件

DataFrame.to_excel() - 将 DataFrame 写入 Excel 文件，to_excel() 方法用于将 DataFrame 写入 Excel 文件，支持 .xls 和 .xlsx 格式。
语法格式如下：

1

DataFrame.to_excel(excel_writer, *, sheet_name='Sheet1', na_rep='', float_format=None, columns=None, header=True, index=True, index_label=None, startrow=0, startcol=0, engine=None, merge_cells=True, inf_rep='inf', freeze_panes=None, storage_options=None, engine_kwargs=None)

部分参数： 参数说明：

  • excel_writer：这是必需的参数，指定了要写入的 Excel 文件路径或文件对象。
  • sheet_name=‘Sheet1’：指定写入的工作表名称，默认为 ‘Sheet1’。
  • na_rep=’’：指定在 Excel 文件中表示缺失值（NaN）的字符串，默认为空字符串。
  • float_format=None：指定浮点数的格式。如果为 None，则使用 Excel 的默认格式。
  • columns=None：指定要写入的列。如果为 None，则写入所有列。
  • header=True：指定是否写入列名作为第一行。如果为 False，则不写入列名。
  • index=True：指定是否写入索引作为第一列。如果为 False，则不写入索引。
  • index_label=None：指定索引列的标签。如果为 None，则不写入索引标签。
  • startrow=0：指定开始写入的行号，默认从第0行开始。
  • startcol=0：指定开始写入的列号，默认从第0列开始。
  • engine=None：指定写入 Excel 文件时使用的引擎，默认为 None，pandas 会自动选择。
  • merge_cells=True：指定是否合并单元格。如果为 True，则合并具有相同值的单元格。
  • inf_rep=‘inf’：指定在 Excel 文件中表示无穷大值的字符串，默认为 ‘inf’。
  • freeze_panes=None：指定冻结窗格的位置。如果为 None，则不冻结窗格。
  • storage_options=None：用于云存储的参数字典。
  • engine_kwargs=None：传递给引擎的额外参数字典。

import pandas as pd
from openpyxl import load_workbook

def read_excel(excel_href):
    # openpyxl读取
    wb = load_workbook(excel_href)
    ws = wb.active
    # pandas读取
    pd_re = pd.read_excel(excel_href, engine="openpyxl")
    return wb, ws, pd_re

def row_update(f_excel_href,insert_data,col_index_str, col_index_num):
    # 调用读取表格函数
    title_wb,title_ws, title_pd_re = read_excel(f_excel_href)

    # 2. 查找书名列的最后一行的行号
    last_row = title_ws.max_row  # 获取当前工作表的最后一行
    for row in range(last_row, 0, -1):  # 从最后一行向上查找
        if title_ws[f"{col_index_str}{row}"].value:
            last_book_row = row  # 记录最后一本书的行号
            break

    for num, l in enumerate(insert_data):
        title_ws.cell(row=last_book_row, column=col_index_num + num + 1, value=l)  # num索引从0开始

    title_wb.save(r"C:\Users\ASUS\Desktop\new.xlsx")

    return title_wb

def col_update(f_excel_href,insert_data,col_index_str, col_index_num):
    # 调用读取表格函数
    title_wb,title_ws, title_pd_re = read_excel(f_excel_href)
    # 2. 查找书名列的最后一行的行号（假设书名在 E 列）
    last_row = title_ws.max_row  # 获取当前工作表的最后一行
    for row in range(last_row, 0, -1):  # 从最后一行向上查找
        if title_ws[f"{col_index_str}{row}"].value:  # 如果 E 列当前行有数据
            last_book_row = row  # 记录最后一本书的行号
            break

    for num,l in enumerate(insert_data):
        title_ws.cell(row=last_book_row + num , column=col_index_num , value=l)  # num索引从0开始
    title_wb.save(r"C:\Users\ASUS\Desktop\new.xlsx")
