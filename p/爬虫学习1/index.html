写爬虫时最好使用定义函数，将每一部分功能都分到不同的函数中封装起来，需要使用时互相调用，这样有助于后期管理，代码也会更清晰

概念和第三方库下载

概念

爬虫：简单来说就是一段自动抓取互联网信息的程序，从互联网上抓取对于我们有价值的信息。
爬虫需要的部分至少有:

  • 请求部分：获取网页源代码
  • 内容解析部分：通过BeautifulSoup、正则表达式等方法对自己需要的部分进行定位截取
  • 保存部分：可以通过os将爬取到的内容保存至文件夹中

其余的个部分功能基本都是在上述三模块后面进行完善

第三方库下载

写爬虫时会使用到一些第三方库，这些第三方库不是可以直接调用的

  • 以Beautiful Soup安装为例：

我们需要打开cmd，进入到 Python（Python2.7版本）安装目录中的 scripts 下，输入 dir 查看是否有 pip.exe, 如果用就可以使用 Python 自带的 pip 命令进行安装，输入以下命令进行安装即可：

1

pip install beautifulsoup4

也可以在pycharm的终端中直接下载

  • 测试是否安装成功： 编写一个python文件，输入
1
2

import bs4
print bs4

如果正常输出则安装成功

(1) 定义请求函数

首先要导入requests库（需要安装），然后使用requests.get方法获取
为了防止一些网页有反爬，需要将请求头headers进行一下伪装
最后定义编码格式并进行返回，完整代码如下

def get_response(html_url):    # 函数参数(html_url)：要请求的链接（url）
    # 定义请求头
    headers = {
        'User-Agent': 'Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/122.0.0.0 Safari/537.36 Edg/122.0.0.0'
    }
    # 获取小说源代码
    html_data = requests.get(url=html_url, headers=headers)
    # 定义编码格式
    html_data.encoding = html_data.apparent_encoding
    # 返回源代码
    return html_data

(2) 获取书名

获取到网站源码以后，接下来姐可以按需要获取所需要的信息了，这里可以选择各种方式，parsrl、正则表达式、BeautifulSoup4等
根据前面获取到的信息，可以发现书名都在

标签下的子级标签中，每本书的a标签都对应着不同的书号，这里我选择用正则表达式进行匹配，首先需要导入正则表达式（re）模块，后续代码写法如下：

1

r'<dt>.*?<a\b[^>]*>(.*?)</a>.*?</dt>'

其中

<dt> - 匹配 <dt> 标签
.*? - 惰性匹配 <dt> 开始标签和 <a> 标签之间的任意内容
<a\b[^>]*> - 匹配 <a 及其可能的属性（如 href="…"）
(.*?) - ​​捕获组​​：匹配 <a> 标签内的文本内容
.*?</dt> - 惰性匹配 </a> 和 </dt> 之间的任意内容，并确保 </dt> 闭合

完整函数如下

def get_data(html_url_2):   # 函数参数(html_url_2)： 小说目录页链接
    # 获取目录页源代码
    html_data = get_response(html_url_2).text
    # 获取小说名字)
    name = re.findall(r'<dt>.*?<a\b[^>]*>(.*?)</a>.*?</dt>', html_data)
    return name

for i in book_name:
    print(i)

(3) 将结果保存到本地

获取到书名后，可以将书名存入txt并保存到文本，这里只演示最简单的保存，后续还可以用os等文件夹操作方法将书名存入自定义文件夹

定义保存函数

1
2
3

def save(name):
    with open('书名.txt', mode='a', encoding='utf-8') as f:
        f.write(name)

短短几行代码就可以将书名保存下来，下面调用函数：

1
2
3
4

book_name = get_data("https://www.xbqg99.cc/")
print(book_name)
for i in book_name:
    save(i + "\n")
